Basic data types in C
    เป็นการบอกกับคอมพิวเตอร์ว่า ข้อมูลที่เราต้องการจะเก็บคืออะไร และต้องการใช้พื้นที่เท่าไหร่
    โดยมีประเภทสำหรับจัดเก็บดังนี้
        1. int      
            -> มีขนาด 4 (32-bit) หรือ 6 (64-bit) ไบต์ ขึ้นอยู่บกับเครื่องคอมพิวเตอร์
            -> จัดเก็บเลขจำนวนเต็มบวกและเต็มลบ
            -> สามารถ assign hex number ได้ เช่น int rgbColor = 0xFFEF0D

        2. float    
            -> จัดเก็บเลขจำนวนเต็มที่มีทศนิยม
            -> มีขนาด 4 ไบต์ 

        3. double   
            -> จัดเก็บเลขจำนวนเต็มที่มีทศนิยม
            -> มีขนาด 8 ไบต์

        4. char     
            -> 

        5. _Bool    
            -> จัดเก็บค่าจริง/เท็ต || true / false || 1 / 0


    _Bool
        สาเหตุที่ boolean จะต้องมี _ นำหน้า เนื่องจากมาตฐานท่แตกต่างกัน
        โดยในมาตรฐาน C89 จะต้องมี _ 
        แต่ในมาตรฐาน C99 จะใช้ bool โดยจะต้อง include #include <stdbool.h>

    โดยในภาษา C จะมีชนิดตัวแปรประเภทอื่น ๆ อีก 
    เช่น short, long, unsigned
    ตัวอย่างของการนำมาใช้
        - short int 
        - long int

    ยิ่งเราใช้ตัวแปรที่ขนาดเหมาะสมมากเท่าไหร่ ก็จะใช้ memory น้อย

Variable and Data types
    โปรแกรมจำเป็นจะต้องจัดเก็บข้อมูลไว้บนหน่วยความจำในขณะที่มันกำลังดำเนินการ

    เมื่อโปรแกรมต้องการส่งชุดคำสั่งไปให้ CPU

    RAM (Random Access Memory)
        - ใช้จัดเก็บข้อมูลของตัวแปร
        - สามารถเข้าถึงข้อมูลได้อย่างรวดเร็ว
        - ข้อมูลจะหายไปเมือคอมพิวเตอร์หยุดทำงาน
        - จัดเก็บอยู่ในรูปแบบฐาน 16

    HDD (Hard Disk Drive)
        - ใช้จัดเก็บข้อมูลถาวร ข้อมูลจะยังอยู่แม้คอมพิวเตอร์จะปิดอยู่
        - จัดเก็บเลขฐาน 2 (1-0)

    เมื่อเราจัดเก็บข้อมูลไว้บนอุปกรณ์ฮาร์ดแวร์ เราจะต้องทราบที่อยู่ของข้อมูล บนอุปกรณ์ฮาร์ดแวร์นั้น ๆ

    * constants เป็นข้อมูลที่ไม่สามารถเปลี่ยนแปรงได้ ระหว่างการทำงานของโปรแกรม
    * variable เป็นข้อมูลที่สามารถเปลี่ยนแปลงได้ ระหว่างการทำงานของโปรแกรม

    * constants และ variable เป็นการจัดเก็บข้อมูลบนหน่วยความจำ

    ::: variable
        เป็นตัวแทนของข้อมูล โดยจะจัดเก็บที่อยู่ของข้อมูลบน ram ไว้ ซึ่งทำให้เราสามารถเข้าถึงและแก้ไข
        ข้อมูลได้อย่างง่าย

    กฏการตั้งชื่อของตัวแปร
        - จะต้องขึ้นต้นด้วยตัวเล็ก (camel case)
        - สามารถใช้ underscore ( _ )
        - สามารถใช้ตัวเลข (0-9)
        - ไม่สามารถขึ้นต้นด้วยตัวเลขได้
        - ไม่สามารถใช้คำศัพท์เฉพาะได้
    ตัวอย่าง
        - jason
        - myFlag
        - i
        - j5x7
        - my_data
        - _anotherVersion

    ::: data types
        ประเภทของข้อมูลที่จะจัดเก็บ โดยสาเหตุที่เราจะต้องระบุชนิดข้อมูล เพราะว่า เราจะต้องรู้ว่า
        ข้อมูลที่เราจะต้องจัดเก็บนั้นเป็นแบบไหน เช่น ตัวเลข หรือ ข้อความ และจะต้องใช้กี่ไบต์ในการจัดเก็บ
        โดยในภาษา c จะมีชนิดข้อมูลดึงนี้
            1.  
            2. 
            3. 
            4. 
            5.

    ::: declaring variable
        - type-specifier variable-name;
        - int age;
    
    ::: assign variable 
        - = value;
        - = 10;

    ::: initialize variable 
        - type-specifier variable-name = value;

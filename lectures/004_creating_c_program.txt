Creating c program
    เราจะมี 4 ขั้นตอนดังนี้
        1. Editing
            เป็นขั้นตอนการเขียนโปรแกรมขึ้นมา

        2. Compiling
            แปลงภาษาคอมพิวเตอร์ระดับสูงเป็นภาษาเครื่อง
            โดยในขั้นตอนนี้จะทำการตรวจสอบชุดคำสั่งด้วยว่ามีข้อผิดพลาดหรือไม่ (syntax error)
            โดยเราสามารถ compile program ได้ด้วยคำสั่ง gcc <file_name.c>

        3. Linking
            เป็นขั้นตอนที่จะเกิดขึ้นหลังจากที่โปรแกรมของเรา compile เรียบร้อยแล้ว 
            โดยจะเป็นการนำ library ต่าง ๆ ที่โปรแกรมของเราใช้ นำมารวมไว้ในโปรแกรมเดียว

        4. Executing
            โปรการเรียกใช้งานโปรแกรมที่เป็นภาษาเครื่องแล้ว

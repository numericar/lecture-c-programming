Comment Code
    ใช้เพื่อจัดทำเอกสารภายในโปรแกรม เพื่อเพิ่มความสามารถในการอ่าน (ทำให้อ่านง่ายขึ้น)
    เช่น เราอาจให้รายละเอียดว่า ชุดคำสั่งน้กำลังทำอะไร

    โดยส่วนที่เป็น comment จะไม่ถูก compile โดย compiler
    
    โดยจะมีการ comment อยู่ 2 แบบ
        1. //       -> ใช้ comment บรรทัดเดียว
        2. /* */    -> ใช้ comment หลายบรรทัด

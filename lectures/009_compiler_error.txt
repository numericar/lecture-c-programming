Complier Errors
    ในขั้นตอนการ compile จะมีการตรวจสอบไวยกรณ์ของโค้ดทั้งหมด
    โดยถ้าถูกต้องทั้งหมด ไม่มีข้อผิดพลาดใด ๆ จะทำการสร้างชุดคำสั่งของเครื่อง 
    (machine language)

    โดย compile error จะมีอยู่ 2 แบบ คือ 
        1. Syntax error 
        2. Semantic error 
    
    Syntax error 
        เป็นข้อผิดพลาดที่จะเกิดขึ้น เมื่อเราเขียนโครงสร้างของชุดคำสั่งผิด 
        เช่น ไม่ได้ใสวงเล็บ, ไม่มีเครื่องหมาย ; ปิดท้ายชุดคำสั่ง

    Semantic error 
        เป็นข้อผืดพลาดที่เขียนถูกต้องตามไวยกรณ์ แต่ไม่ถูกต้องตามเจตนาของโปรแกรม
        เช่น เรามีฟังก์ชัน add(a, b) ทำหน้าที่ีับค่า a และ b มา + กัน แต่เรากลับเขียนคำสั่ง a - b
        ซึ่งไม่ถูกต้อง add(a, b) => a - b;

        และอาจเกิดขึ่นจาก ชนิดของข้อมูล เช่น 
        int x = 5, y = 2
        float result = x / y;
        จากชุดคำสั่งดังกล่าว จะทำงานไม่ถูกต้องเนื่องจากเราจะได้ค่า 2.0 มาแทนที่จะเป็น 2.5
        โดยเกิดจากข้อผิดพลาดที่เราไม่ทำการแปลงชนิดข้อมูลก่อน

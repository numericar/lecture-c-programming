Fundamantal of a program
    
    ตอมพิวเตอร์นั้นไม่ฉลาด แต่สามารถคำนวนค่าต่าง ๆ ได้อย่างรวดเร็ว
    โดยสิ่งที่เราสามารถทำได้ คือการสร้างคำสั่งขึ้นมาเพื่อสั่งคอมพิวเตอร์ว่าจะต้องทำอะไร 
    เราเรียกสิ่งนั้นว่า program

    โดยข้างใน program จะมีชุดคำสั่งที่จะทำงานกับ CPU

    โดยเราสามารถเขียนชุดคำสั่งได้จากภาษาคอมพิวเตอร์ต่าง ๆ ที่ถูกสร้างขึ้น 
    เช่น 
        - c 
        - c++ 
        - c# 
        - java

    โดยภาษาเหล่านี้จะทำหน้าที่นำชุดของคำสั่งที่เราเขียน ไปสั่งงาน CPU

Higher level programming language
    เป็นภาษาคอมพิวเตอร์ระดับสูงที่ทำให้สามารถเขียนได้ง่าย มนุษย์อ่านแล้วเข้าใจ

Complier
    ทำหน้าที่แปรงภาษาคอมพิวเตอร์ระดับสูง (เช่น c, java) ให้เป็นภาษาที่เครื่องสามารถอ่านเข้าใจได้
    เช่น assembly

โดยเรามีขั้นตอนการพัฒนาโปรแกรมดังนี้
    1. define the program objective
    2. design the program 
    3. write the code 
    4. compile 
    5. run the program 
    6. test and debug program 
    7. maintaun and modufy the program
